# 🧙‍♂️ 霍格華茲氣象報告 - 宜蘭分校 

這是一個充滿魔法氛圍的個人天氣預報網站。我們將麻瓜世界的氣象數據，轉化為霍格華茲風格的魔法報告，讓查看天氣變得像閱讀《預言家日報》一樣有趣。

## 📂 專案結構

本專案採用純靜態網頁結構，無需複雜的建置工具即可運行。

```text
VIBECODING-CWAWEATHER-FRONTEND/
├── images/               # 🪄 魔法素材庫
│   ├── cursor-wand.png   # 魔杖游標圖片
│   ├── icon-bg.jpg       # 霍格華茲城堡背景圖
│   └── icon-v2.png       # 網站圖示 (Favicon)
├── index.html            # 🔮 主要入口檔案 (HTML/CSS/JS)
└── .gitignore            # Git 設定檔
```

## ✨ 特色功能

- 沉浸式魔法介面：
  - 使用霍格華茲城堡作為背景。
  - 羊皮紙風格的資訊卡片。
  - 魔杖游標：瀏覽網頁時，滑鼠會化身為魔杖。

- 互動視差效果：滑鼠懸停在天氣卡片上時，會產生如同咒語般的漂浮 3D 特效。

- 魔法生活建議：

  - 將降雨機率轉化為「需不需要施展防水咒」的建議。
  - 將氣溫轉化為「巫師袍穿搭指南」。

- 即時資料串接：

  - 串接中央氣象署 API (進階呼嚕粉可連通麻瓜世界網域)。
  - 載入時顯示「劫盜地圖」風格的 Loading 動畫。

## 🚀 如何使用

- 確認 images 資料夾內已備妥以下圖片素材：

  - cursor-wand.png
  - icon-bg.jpg
  - icon-v2.png

- 直接使用瀏覽器 (Chrome, Edge, Safari 等) 開啟 index.html 即可看到魔法天氣報告。


## 🛠️ 技術說明

- HTML5 / CSS3 / Vanilla JavaScript：無使用任何框架，輕量快速。
- Google Fonts：使用 Zen Maru Gothic 字體，營造圓潤且帶有手寫感的魔法風格。
- RWD 響應式設計：支援手機與電腦版面瀏覽。
